# 📊 Retail Sales Analysis

## Overview

This project analyzes over **51,000 retail transactions** from a global retail company using SQL, Excel, and Power BI.

The objective is to identify sales trends, customer behavior, profitable products, and business opportunities through data-driven analysis.

## Business Problem

The management team wants answers to questions such as:

- Which regions generate the highest revenue?
- Which products are the most profitable?
- Who are the top customers?
- How do discounts affect profit?
- Which categories require strategic attention?

## Dataset

- Global Superstore Dataset
- Records: **51,290**
- Features: **27**
